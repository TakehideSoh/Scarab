Not

Related Doc: package scarab

case class Not(p: Bool) extends Literal with Product with Serializable

Not is a case class for negations of Boolean variables.

Prefix operator ! defined in Term class can be used to create this object.

scala> val p = Bool("p")
scala> Not(p)               // Not(p)
scala> ! p                  // Not(p)
p
